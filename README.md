# 房屋租赁管理系统 java61

## 项目概述

房屋租赁管理系统是一款基于B/S架构的Web应用，采用前后端分离的设计模式，结合MVC架构，为用户提供了一个便捷、高效的管理平台。系统主要使用Java和Vue技术栈开发，集成了SpringBoot和Mybatis等框架。

## 技术栈

- 架构: B/S、MVC
- 系统环境：Windows/Mac
- 开发环境：
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql
- 技术栈：
  - Java
  - Mysql
  - SpringBoot
  - Mybatis
  - Vue

## 功能模块

### 系统角色

#### 管理员

- 公告信息管理
  - 发布公告
  - 编辑公告
  - 删除公告
- 房屋信息管理
  - 添加房屋
  - 编辑房屋信息
  - 删除房屋
- 用户信息管理
  - 查看用户
  - 编辑用户信息
  - 管理用户权限
- 租金信息管理
  - 设置租金标准
  - 编辑租金
  - 查看租金记录
- 故障信息管理
  - 受理故障申报
  - 派遣维修人员
  - 记录维修情况

#### 用户

- 查看公告
- 房屋信息查询
- 租金查询
- 故障申报

## 运行环境

系统可运行于以下环境：

- 服务器：任何支持Java和Mysql的服务器环境
- 客户端：主流现代浏览器

## 部署步骤

1. 安装Java JDK 1.8
2. 安装Mysql数据库，创建数据库及用户
3. 使用Maven构建项目
4. 将项目部署至Web服务器
5. 访问系统进行配置和管理

## 核心亮点

- 前后端分离设计，便于维护和扩展
- 基于SpringBoot框架，提高开发效率
- 采用Mybatis作为持久化框架，灵活映射数据库操作
- 清晰的目录结构，便于团队协作开发
- 集成万字文档，方便用户学习和使用系统
